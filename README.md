# Одностраничный сайт для автосалон на примере Mersede-Benz A 200

На сайте реализовано:
+ Аккордион
+ Плавная прокрутка 
+ Переключение табов и смена содержимого
+ Модальное окно
+ Бургер-меню в мобильной версии
